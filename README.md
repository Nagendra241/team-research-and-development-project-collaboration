# # Countries of the World: GDP vs Infant Mortality

This project explores whether there is a statistical correlation between *GDP per capita* and *infant mortality rates* across countries. It forms part of a team-based research and development initiative focused on global health and economic indicators.
We hypothesize that countries with higher GDP per capita tend to have lower infant mortality rates, reflecting better access to healthcare, nutrition, and infrastructure.
